# Real-Time-Face-Eye
Real-time face and eye detection system using OpenCV and Haar Cascade classifiers in Python.

## Description
This project is a real-time Face and Eye Detection system build Python and OpenCV.
It uses pre-trained Haar Cascade classifiers to detect faces and eyes from a live webcam feed.
This system captures video input, processes each frame, and highlights detected faces and eyes with bounding boxes. 
This project is ideal for beginners in Camputer Vision.

## Tech Stack
python
OpenCV
NumPy
